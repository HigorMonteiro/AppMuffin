# AppMuffin

Python 3.7

## Description

Example App using micro-framework Muffin

## Download aplication


Step by step

```bash
$ git clone https://github.com/HigorMonteiro/AppMuffin
$ cd AppMuffin
$ virtualenv .env -p /usr/bin/python3.5
$ source .venv/bin/activate
$ pip install -r requirements.txt
$ muffin app run
```
